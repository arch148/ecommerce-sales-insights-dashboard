# E-Commerce Sales Insights Dashboard

## Project Overview
This project is an end-to-end e-commerce data analytics project built using Python, SQL, and Power BI to analyze sales performance, customer behavior, delivery metrics, and product insights.

## Tools & Technologies
- Python
- SQL (MySQL)
- Power BI
- DAX
- Excel

## Project Features
- Sales revenue analysis
- Customer behavior insights
- Repeat customer tracking
- Delivery performance analysis
- Product category performance analysis
- Order status distribution analysis
- State-wise revenue analysis
- Interactive Power BI dashboard

## Key KPIs
- Total Orders: 99K
- Total Customers: 96K
- Total Revenue: 16.01M
- Avg Delivery Days: 12.50
- Repeat Customers: 3K
